class Solution:

    def romanToInt(self, s: str) -> int:

        values = [1, 5, 10, 50, 100, 500, 1000]

        chars = ['I', 'V', 'X', 'L', 'C', 'D', 'M']

        map = {char: value for char, value in zip(chars, values)}

        total = 0

        prev_value = 0

        for char in reversed(s):

            current_value = map[char]

            if current_value < prev_value:

                total -= current_value

            else:

                total += current_value

            prev_value = current_value

        return total

  

solution = Solution()

print(solution.romanToInt("III"))    

print(solution.romanToInt("LVIII"))  

print(solution.romanToInt("MCMXCIV"))

### Описание кода:

1. **Класс и метод**: В коде создается класс, который содержит метод для преобразования римских чисел в целые числа. Метод принимает строку, представляющую римское число, и возвращает его числовое значение.
    
2. **Определение значений**: Внутри метода создаются два списка: один содержит значения римских символов (например, 1 для "I", 5 для "V" и так далее), а другой — сами римские символы. Затем создается словарь, который связывает каждый римский символ с его числовым значением.
    
3. **Инициализация переменных**: Перед началом обработки римского числа инициализируются переменные для хранения итогового значения и значения предыдущего символа.
    
4. **Обработка строки**: Метод проходит по символам римского числа в обратном порядке. Это позволяет легче определять, когда нужно вычитать значение (например, в случаях, когда меньшее значение предшествует большему).
    
5. **Логика вычислений**: Для каждого символа определяется его числовое значение. Если текущее значение меньше значения предыдущего символа, оно вычитается из итогового значения. В противном случае текущее значение добавляется к итоговому значению.
    
6. **Обновление предыдущего значения**: После обработки каждого символа обновляется значение предыдущего символа, чтобы оно могло быть использовано в следующей итерации.
    
7. **Возврат результата**: После обработки всех символов метод возвращает итоговое числовое значение, которое соответствует введенному римскому числу.
    

### Примеры использования:

В конце кода приводятся примеры, где метод вызывается с различными римскими числами, и выводится их числовое представление. Например, "III" преобразуется в 3, "LVIII" в 58, а "MCMXCIV" в 1994.
